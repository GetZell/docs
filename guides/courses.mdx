## Overview

Courses provide structured learning paths for your team with multiple modules, text content, and agent practice sessions. Each course has an **intensity preset** that provides the default number of practice calls required and the minimum score a call must reach to count.

## Intensity Presets

When creating a course, you select one of three intensity presets that define the training rigor:

| Preset | Calls per Exercise | Min. Score per Call | Best For |
|--------|-------------------|---------------------|----------|
| **Light** | 2 calls | ≥6/10 | Refreshers, product updates, quick check-ins |
| **Standard** | 5 calls | ≥7/10 | Ongoing training, skill building |
| **Intensive** | 10 calls | ≥8/10 | New hire onboarding, certification programs |

<Note>Intensity presets set the default call requirement and score threshold for agent call steps. Admins can override the required calls or pass threshold on an individual agent call step when a specific exercise needs different requirements.</Note>

## Creating a Course

Course creation is a 2-step process: first configure the course details, then build the module structure.

### Step 1: Course Details

<Steps>
  <Step title="Navigate to Courses">
    Go to **Team Management** → **Courses** in the main navigation and click **Create Course**.
  </Step>

  <Step title="Set Title & Description">
    Configure the basic course information:

    - **Title:** Enter a clear, descriptive course name (minimum 3 characters)
    - **Description:** Summarize what learners will achieve (minimum 3 characters)

    **Example:**
    - Title: "Sales Fundamentals Bootcamp"
    - Description: "Master consultative selling through interactive lessons and practice sessions with AI prospects."
  </Step>

  <Step title="Select Intensity Preset">
    Choose the training intensity that matches your course goals:

    - **Light:** Quick refreshers with 2 calls per exercise, each call must score ≥6/10
    - **Standard:** Regular training with 5 calls per exercise, each call must score ≥7/10
    - **Intensive:** Rigorous programs with 10 calls per exercise, each call must score ≥8/10

    <Tip>Start new hires with **Intensive** courses for thorough onboarding, and use **Light** for periodic product updates.</Tip>
  </Step>

  <Step title="Select Target Groups (Optional)">
    Choose which groups this course is designed for. This filters the available agents and users to only show those relevant to the selected groups.

    **Why use target groups?**
    - Focus on specific teams or departments
    - Filter agents trained for those teams
    - Pre-filter users when enrolling
  </Step>

  <Step title="Enroll Users">
    Select which team members should be enrolled in this course. If you selected target groups, only users from those groups will be shown.

    <Note>You can add or remove users later directly from the course card—no need to edit the course.</Note>
  </Step>

  <Step title="Continue to Modules">
    Click **Next** to proceed to building your course modules.
  </Step>
</Steps>

### Step 2: Build Modules

<Steps>
  <Step title="Add Modules">
    Click **Add Module** for each major topic in your course. Each module is a collapsible section containing related steps.

    For each module, provide:
    - **Module Title:** The name of this section (e.g., "Discovery Fundamentals")
    - **Module Description:** What participants will learn in this module

    <Note>Modules appear in the order you create them. Click on a module to expand or collapse its steps.</Note>
  </Step>

  <Step title="Add Steps to Each Module">
    Inside each module, click **Add Step** and choose the step type:

    **Text Step:** Educational content, instructions, or reference materials
    - **Title:** Name of the step
    - **Description:** Optional additional context
    - **Content:** The actual text content for learners to read

    **Agent Call Step:** Practice sessions with AI agents
    - **Title:** Name of the practice session
    - **Agent:** Select which bot learners will practice with (filtered by target groups if selected)
    - **Calls Required:** Optional override for this practice step. If left empty, the course intensity preset is used.
    - **Pass Threshold:** Optional score threshold for this practice step. If unchanged, the course intensity preset is used.

    <Info>When no step-level override is set, the required calls come from the course's intensity preset. For example, a **Standard** course requires 5 calls per agent step.</Info>

    <Tip>Alternate between text steps (learning) and agent call steps (practice) for better retention.</Tip>
  </Step>

  <Step title="Save Course">
    Click **Create** to save the course and make it available to enrolled users.
  </Step>
</Steps>

## Step Types

### Text Step

Text steps deliver written educational content to learners.

**Fields:**
- **Title:** Name of the step (required, minimum 3 characters)
- **Description:** Optional additional context
- **Content:** The actual text content learners will read

**Use Cases:**
- Introduction to concepts
- Theory and frameworks
- Best practices guides
- Scripts and talk tracks
- Reference materials

### Agent Call Step

Agent call steps require learners to complete practice conversations with AI bots. The number of calls and the minimum score per call use the course's intensity preset by default, unless an admin sets step-specific requirements.

**Fields:**
- **Title:** Name of the practice session (required, minimum 3 characters)
- **Agent:** Select which bot learners will practice with (required)
- **Calls Required:** Optional step-specific number of passing calls
- **Pass Threshold:** Optional step-specific minimum score from 0 to 10

**Pass Criteria:**
- Each call must score at or above the step's threshold to count toward completion
- Accumulate the required number of passing calls for the step

**Use Cases:**
- Applying learned techniques
- Building muscle memory
- Handling specific scenarios
- Skill practice and assessment

## Managing Enrollment

### Adding Users to a Course

You can add users directly from the course card without editing the course:

<Steps>
  <Step title="Expand Enrolled Users">
    On any course card, click **Enrolled People** to expand the enrolled users section.
  </Step>

  <Step title="Click Add User">
    Click the **+ Add User** button at the top of the enrollment list.
  </Step>

  <Step title="Select User">
    Search for and select the user you want to enroll. They'll be added immediately.
  </Step>
</Steps>

### Removing Users from a Course

<Steps>
  <Step title="Expand Enrolled Users">
    On the course card, expand the enrolled users section.
  </Step>

  <Step title="Remove User">
    Hover over the user row and click the **X** button that appears to remove them from the course.
  </Step>
</Steps>

<Note>Removing a user from a course does not delete their progress—if re-enrolled, their progress will be restored.</Note>

## Example Course Structure

Here's a complete example of a well-structured course:

<Accordion title="Sales Fundamentals Bootcamp - Standard Intensity">

**Intensity:** Standard (5 passing calls per exercise, each call must score ≥7/10)

**Module 1: Discovery Fundamentals**
1. Text Step: "Introduction to Consultative Selling"
2. Text Step: "The SPIN Questioning Framework"
3. Agent Call Step: Practice discovery calls (5 passing calls with "Friendly Prospect")

**Module 2: Qualifying Prospects**
1. Text Step: "BANT Qualification Framework"
2. Text Step: "Reading Buying Signals"
3. Agent Call Step: Qualification practice (5 passing calls with "Mixed Signals Prospect")

**Module 3: Presenting Solutions**
1. Text Step: "Features vs Benefits"
2. Text Step: "Tailoring Your Pitch"
3. Agent Call Step: Pitch practice (5 passing calls with "Technical Buyer")

**Module 4: Handling Objections**
1. Text Step: "Common Objections and Responses"
2. Text Step: "The Feel-Felt-Found Method"
3. Agent Call Step: Objection handling drills (5 passing calls with "Skeptical Prospect")

**Module 5: Closing Techniques**
1. Text Step: "Recognizing Buying Signals"
2. Text Step: "Trial Closes and Assumptive Closes"
3. Agent Call Step: Full cycle role plays (5 passing calls with "Ready-to-Buy Prospect")

</Accordion>

## Managing Courses

### Viewing Courses

Navigate to **Team Management** → **Courses** to see all courses with:
- Course title, description, and intensity preset
- Number of enrolled users
- Total steps across all modules
- Overall progress percentage
- Enrolled user details and individual progress

### Editing a Course

To update an existing course:
1. Click the **Edit** icon on the course
2. Update course details:
   - Title
   - Description
   - Intensity preset
3. Save your changes

<Note>Changes to the intensity preset affect agent call steps that do not have step-specific call requirements or pass thresholds. User enrollment is managed directly on course cards.</Note>

### Tracking Progress

For each course, you can monitor:
- **Enrollment count:** How many users are enrolled
- **Overall progress:** Average completion percentage across all users
- **Individual progress:** Each user's completion status and average score
- **Completed steps:** Number of steps each user has finished
- **Module completion:** Which modules users have completed

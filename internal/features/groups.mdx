## Overview

Groups organize users with bots and personas so teams can receive targeted training access. Users can belong to multiple groups, and groups can simplify assignment management.

## Key Concepts

<CardGroup cols={2}>
  <Card title="User Membership" icon="user-plus">
    Users can be added to groups to receive access to assigned resources.
  </Card>

  <Card title="Bot Assignment" icon="bot">
    Bots assigned to a group become available to users in that group.
  </Card>

  <Card title="Persona Assignment" icon="user-round-cog">
    Personas can be assigned to groups for shared simulation behavior.
  </Card>

  <Card title="Multiple Groups" icon="layers">
    Users and resources can participate in multiple groups.
  </Card>
</CardGroup>

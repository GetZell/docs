## Overview

Settings control tenant-level and user-level configuration such as language, provider integrations, account defaults, and application preferences.

## Key Concepts

<CardGroup cols={2}>
  <Card title="Tenant Settings" icon="building">
    Account-level configuration shared across the tenant.
  </Card>

  <Card title="User Settings" icon="user-cog">
    Individual user preferences and profile-level configuration.
  </Card>

  <Card title="Language Settings" icon="languages">
    Locale and language settings affect UI, notifications, and localized call content.
  </Card>

  <Card title="Provider Settings" icon="plug">
    Integrations such as Aircall or other call providers can have dedicated settings pages.
  </Card>
</CardGroup>

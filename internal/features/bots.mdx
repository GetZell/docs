## Overview

Bots are AI agents used for training conversations. They define the simulated counterpart, scenario behavior, voice configuration, language, and scoring/evaluation setup used in practice calls.

## Key Concepts

<CardGroup cols={2}>
  <Card title="Agent" icon="bot">
    The configured AI training counterpart used in simulated calls.
  </Card>

  <Card title="Scenario" icon="scroll-text">
    The sales, support, hiring, or practice situation the bot should simulate.
  </Card>

  <Card title="Voice & Language" icon="mic">
    Bot calls can be configured with voices, languages, and transcriber settings.
  </Card>

  <Card title="Scorecard Assignment" icon="star">
    Bots connect to scorecards so calls can be evaluated consistently.
  </Card>

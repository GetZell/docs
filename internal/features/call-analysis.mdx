## Overview

Call Analysis is the detail view for a single call. It combines recording playback, transcript context, evaluation results, feedback comments, inline coaching, and navigation from snippets or history.

## Key Concepts

<CardGroup cols={2}>
  <Card title="Recording" icon="audio-lines">
    Audio playback is available when the backend can resolve a recording URL.
  </Card>

  <Card title="Transcript" icon="file-text">
    Transcript messages provide the context for scorecards, feedback, and snippets.
  </Card>

  <Card title="Evaluation" icon="star">
    Calls can include scorecard score, Zell score, categories, and signal-level results.
  </Card>

  <Card title="Inline Coaching" icon="message-circle">
    Feedback threads can be anchored to transcript positions and opened from inbox or history flows.
  </Card>
</CardGroup>

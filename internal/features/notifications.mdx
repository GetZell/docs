## Overview

Notifications keep users informed about coaching, streaks, training, and platform events. Backend notification templates and delivery jobs support localized messages and deep links.

## Key Concepts

<CardGroup cols={2}>
  <Card title="Notification Template" icon="file-text">
    Defines message content for a specific notification event.
  </Card>

  <Card title="Delivery Channel" icon="send">
    Notifications can target channels such as email, Slack, or in-app messages.
  </Card>

  <Card title="Localization" icon="languages">
    Notification copy can be localized based on user locale.
  </Card>

  <Card title="Deep Link" icon="link">
    Some notifications include app links that take users to the relevant workflow.
  </Card>
</CardGroup>

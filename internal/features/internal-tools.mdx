## Overview

Internal Tools are restricted surfaces for Zell operators and internal admins. They support tenant management, model observability, default scorecards, voices, bot sentences, playbooks, and user feedback inspection.

## Course Review

Internal tenant course review pages use the same module overview as the admin course view. For agent call steps, the displayed call requirement comes from the step-specific `callGoal` when present. If a step has no override, the view falls back to the course intensity preset: Light requires 2 calls, Standard requires 5 calls, and Intensive requires 10 calls.

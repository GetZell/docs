## Overview

Call Collections let users save calls for later review, coaching, enablement, or team sharing. Collections are used from call and snippet surfaces where a user wants to keep a call as an example.

## Key Concepts

<CardGroup cols={2}>
  <Card title="Collection" icon="folder-open">
    A saved group of calls used for review, coaching, or examples.
  </Card>

  <Card title="Save for Later" icon="bookmark">
    Snippet and inbox surfaces can open the Add to Collection dialog for the current call.
  </Card>

  <Card title="Call Context" icon="phone">
    Collections preserve access to the call detail view, including transcript and feedback context.
  </Card>

  <Card title="Review Workflow" icon="list-checks">
    Collections can support manager review, enablement examples, or follow-up workflows.
  </Card>
</CardGroup>

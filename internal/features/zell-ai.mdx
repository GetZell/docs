## Overview

Zell AI is the assistant experience inside the platform. It helps users ask questions, inspect application data, and work through AI-assisted workflows.

## Key Concepts

<CardGroup cols={2}>
  <Card title="Chat Session" icon="message-circle">

  </Card>

  <Card title="Database Questions" icon="database">

  </Card>

  <Card title="Tool Use" icon="wrench">

  </Card>

  <Card title="Streaming Answers" icon="radio">
